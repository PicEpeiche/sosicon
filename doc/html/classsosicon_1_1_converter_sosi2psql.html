Detailed Description

SOSI to PostgreSQL/PostGIS converter.

If command-line parameter -2psql is specified, this converter will handle the output generation. Produces a PostgreSQL/PostGIS dump file from the SOSI source(s).

Fetch element data fields recursively.

Traverses the SOSI element tree recursively, extracting plain data fields. The field names are stored in the hdr list, and the data values are stored in the row list. The data size value associated wit each entry in hdr is updated to reflect the longest encountered field length.

Parameters
parentThe SOSI (sub)tree to be traversed.
hdrThe fields list (table header).
rowThe record set (table row).
