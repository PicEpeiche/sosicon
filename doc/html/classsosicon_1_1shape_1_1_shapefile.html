Detailed Description

Shapefile implementation.

Wraps all ESRI Shape output files (shp, shx, dbf, prj...) in one class.

Parameters

selectionSOSI element type scheduled for shapefile conversion. Since a shapefile may contain only one geometry type at a time, one must select what element type to extract from the SOSI file.
