Detailed Description

Shapefile record header.

The header for each shapefile record stores the record number and content length for the record. The header have a fixed length of 8 bytes when serialized.

Int32Field sosicon::shape::ShapeElementHeader::mContentLength
private

Length of record contents.

The content length for a record is the length of the record contents section measured in 16-bit words. Each record, therefore, contributes ( 4 + content length ) 16-bit words toward the total length of the file, as stored at Byte 24 in the file header.
