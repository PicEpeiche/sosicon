sosicon::byteOrder::Endianness sosicon::byteOrder::determine ( )

Determines system endianness.

Tests byte-order to see if the program runs on a big endian or a little endian architecture. Flags the byteOrder::endian variable.

Returns
System endianness.
Return values
Endianness::bigBig endian system.
Endianness::littleLittle endian system.
